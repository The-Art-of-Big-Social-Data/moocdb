A couple notes:
  • Username is not for any authentication, it's just to uniquely identify you as an annotator in the database. If you want to see all of your previous annotations, make sure to use the same username
  • The SELECT statement should be something like "SELECT text_id, raw_text FROM moocdb.posts, moocdb.texts where posts.text_id = texts.id and posts.source_id = 1;"
  • We recommend you try out the SELECT statements in MySQLWorkbench, and then just copy over the statement you want. But basically, you must always return EXACTLY (text_id, raw_text) FROM the posts and texts tables
  • For more details on the BRAT Annotation config file, see their webpage
  • The application will return "Application Error" if you don't fill in all this info
